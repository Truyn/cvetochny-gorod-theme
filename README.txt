ЦВЕТОЧНЫЙ ГОРОД — установка

1. В WordPress откройте Внешний вид → Темы → Добавить новую → Загрузить тему.
2. Выберите архив cvetochny-gorod.zip и активируйте тему.
3. Установите и активируйте WooCommerce.
4. Проверьте, что WooCommerce создал страницы: Магазин, Корзина, Оформление заказа, Мой аккаунт.
5. Внешний вид → Настроить → «Цветочный город: Главная»: загрузите фото первого экрана, укажите телефон, адрес и режим работы.
6. Внешний вид → Меню: создайте главное меню и назначьте его области «Главное меню».
7. Создайте категории товаров со следующими ярлыками (slug):
   svadebnye, sladkie, avtorskie, lux, do-2000, do-5000, do-10000
8. Для главной страницы: Настройки → Чтение → отображать статическую страницу; создайте страницу «Главная» и выберите её. Шаблон front-page.php подключится автоматически.

Важно: изображения в архиве — нейтральные заглушки. Замените их своими фотографиями через настройщик и карточки товаров WooCommerce.


ELEMENTOR
========
1. Установите Elementor и WooCommerce.
2. Создайте или откройте страницу «Главная».
3. В атрибутах страницы выберите шаблон «Elementor — на всю ширину».
4. В Настройки → Чтение назначьте эту страницу главной.
5. Нажмите «Редактировать в Elementor». Когда страница собрана Elementor, тема автоматически выводит её вместо встроенного демонстрационного макета.
6. Для страницы без шапки и подвала используйте «Elementor — холст».
